# TV MAIS+

Landing page premium para a marca **TV MAIS+** — streaming de canais ao vivo, filmes, séries e esportes.

## Stack

- React 18 + Vite 5
- Tailwind CSS 3
- Framer Motion
- Lucide React

## Desenvolvimento

```bash
npm install
npm run dev
```

## Build de produção

```bash
npm run build
npm run preview
```

## Deploy

Configurado para Vercel. Domínio: [tvmaisplus.vercel.app](https://tvmaisplus.vercel.app)
